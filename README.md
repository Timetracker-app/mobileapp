## TimeTracker - mobile application

**Technologies and tools:**

- React Native

### Setup:

```bash
npx @react-native-community/cli@latest init timetracker

cd timetracker
npm install
```

**Start the Metro Server:**

The JavaScript _bundler_ that ships _with_ React Native.

```bash
npm start
```

**Start Applicaton:**

```bash
npm run android
```
